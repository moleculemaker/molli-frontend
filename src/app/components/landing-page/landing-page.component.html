Welcome to Molli!

Molli (MOLecular LIbrary) is a chemoinformatics toolkit to manipulate large in silico libraries.

Molli can expand three-dimensional, combinatorial molecule libraries directly from two-dimensional chemical structure fragments stored in CDXML files with high stereochemical fidelity. It supports geometry optimization, property calculation, and conformer generation by interfacing with widely used computational chemistry programs such as OpenBabel, RDKit, ORCA, NWChem, and xTB/CREST.

There are two primary ways to use Molli:

Python Library and Documentation
Install the cross-platform Molli package to take advantage of its powerful command line interface and Python classes for common molecule manipulations, conformer-dependent grid-based feature calculators, and scalable 3D visualizations.
AlhyaSynthesis Web App
Use the point-and-click web application to generate an in silico library of BOX ligands from a set of bases and substituents you provide. Explore the library with interactive tools and export the data for use in other tools.
